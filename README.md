# Chaos Combined

## definition

Chaos Combined is a script combination project that I have undertaken. Currently
there are 5 scripts combined in this script.

## Scripts Included

- Inner Self by LewdLeah
- AutoCards by LewdLeah
- Localized Language by LewdLeah
- Narrative Guidance overhaul by Purplejumps + library addition by myself Dramora9879
- True Auto Stats by Yi1i1i
- ReputeX Engine by Lothens, LewdLeah, MillennialJesus (reputation, world clock, events)

## What Makes This Special Or Unique?

When I started this project all other scripts that combined other scripts were
outdated and needed a specific combination of scripts for a Scenario I had in
mind. In addition to being up-to-date with the other scripts it also has a
semi-unique feature; All current and future scripts have been Integrated into
Inner Self's MainSettings meaning that when you open the Inner Self Config Story
Card you will see options to enable each script separately.

### Current Progress

Currently the scripts isn't even 1% complete I have big plans for Chaos
Combined, maybe even going as far as eventually incorporating all the most
